FeatureId Name Description
1 Single click deployment The whole db schema could be deployed via single click command line batch script
2 Foreign keys handling for ETL The Foreign keys could be stored temporary , removed , data could be loaded and the same foreign keys reapplied
3 Database backup The Database backup could be performed via a single click batch cmd run
4 Export Tables into CSV The database tables could exported into csv files via a single click batch script
5 System wide Global Settings The System has Global Settings, which are stored in a db table and could be changed via run-time
6 User Settings Each user has user settings , each could be changed durning-run time and are inherited from the global settings of the System
7 Generic History table code generation for each table generically could be generated code (triggers and a history table ) for change tracking enabling the answer of the question who has changed what and when per table
